## TESTING SCENARIOS

### Competency Tests (Before Going Fully Independent)

**Test 1: UK Invoice**
- Create invoice for £10,000 race fee
- Add 20% VAT correctly
- Send to UK client
- Record payment

**Test 2: EUR Export Invoice**
- Create €8,000 invoice for German team
- Apply zero-rating correctly
- Check VIES for VAT number
- State reverse charge
- Record EUR payment

**Test 3: Race Weekend Expenses**
- Process flight, hotel, meals, taxi
- Apply correct VAT rates
- Tag with event tracking
- Reconcile to bank

**Test 4: VAT Return**
- Prepare draft VAT return
- Verify all boxes correct
- Explain zero-rated exports
- Identify any issues

**Test 5: Month-End**
- Complete month-end checklist
- Run key reports
- Reconcile all accounts
- Prepare summary for David
